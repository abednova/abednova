### 1. База данных педагогических классов (BSPU PK Base)

**Описание**: Республиканская система мониторинга данных педагогических классов по Республике Беларусь. Веб-приложение для управления и анализа данных об образовательных учреждениях, построенное на Laravel (backend) и Vue 3 (frontend).

**Технологический стек**:
- **Backend**: Laravel 10, PHP 8.2, MySQL, Laravel Passport (OAuth2)
- **Frontend**: Vue 3, TypeScript, Vite, Tailwind CSS, UnoCSS
- **Инфраструктура**: Docker, Docker Compose
- **API**: RESTful API с документацией Swagger/OpenAPI

**Мои задачи**:
- Функциональное тестирование backend и frontend компонентов приложения
- Тестирование REST API через Postman (коллекция из 50+ запросов)
- Написание и поддержка тест-кейсов для 6 основных модулей (1534+ строк детальных тест-кейсов)
- Разработка тестовых сценариев
- Создание чек-листов для быстрой проверки функциональности
- Применение техник тест-дизайна
- Регрессионное тестирование перед релизами
- Документирование баг-репортов (Jira)

**Результаты**:
- Проведено комплексное тестирование веб-приложения (backend - API через Postman, Swagger, БД - MySQL через сторонний клиент, frontend - браузер)
- Разработан пакет тестовой документации
- Выявлены, задокументированы баги влияющие на функциональность системы и после исправления разработчиком повторно проверены и допущены до релиза
- Предложены issues для дальнейшего улучшения проекта
- Обеспечено качество продукта перед каждым релизом посредством регрессионного тестирования

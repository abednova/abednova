- [ ] Фильтрация по району (после выбора региона)
- [ ] Функция сброса фильтров возвращает к исходному состоянию
- [ ] Пересечение фильтров (регион + район) работает
- [ ] Переключатель года влияет на счетчики
- [ ] Переход и отображение карточки школы работает
- [ ] Поиск по ключевому слову
- [ ] Система обрабатывает поиск несуществующих записей
- [ ] Поиск с специальными символами
- [ ] Локализация главной страницы работает
- [ ] Проверка внешних ссылок (ссылки футера)

## Чек-лист 3: Админка - Авторизация

- [ ] Пользователь с валидными учетными данными может успешно войти в административную панель
- [ ] Система выдает ошибку при попытке входа с неверным email
- [ ] Система выдает ошибку при попытке входа с верным email и неверным паролем
- [ ] Система выдает ошибку при попытке входа с пустыми полями
- [ ] Авторизованный пользователь может выйти из системы
- [ ] Неавторизованные пользователи не могут получить доступ к административной панели
- [ ] Ограничение попыток входа (rate limit) 
- [ ] Авторазлогин после idle работает
- [ ] Проверка токена (refresh) 
- [ ] Проверка доступа для разных ролей (manager vs guest) работает

### 2. Сервис интерактивного тестирования на основе Markdown (md-tests.app)

**Описание**: Веб-сервис для создания и прохождения интерактивных тестов на основе Markdown разметки. Проект находится в активной разработке.

**Мои задачи**:
- Функциональное тестирование всех компонентов сервиса:
    - Создание, редактирование и публикация дисциплин и тестов к ним в формате Markdown
    - Тестирование интерактивного прохождения тестов (прохождение теста, проверка результатов, сохранение прогресса)
    - Ролевая система (преподаватель, администратор, студент)
- Написание баг-репортов и их приоритезация
- Написание тестовых артефактов: тест-кейсов, тестовых сценариев, чек-листов для основных функций
- Активное участие в планировании развития проекта: предложения по улучшению UX, обсуждение новых функций

**Результаты**:
- Проведено комплексное функциональное тестирование сервиса
- Выявлены, задокументированы и приоритизированы баги в системе Plane с детальным описанием проблем
- Протестировано и доработано API через Postman
- Все найденные баги устранены разработчиками и был произведен их ретест
- Предложены варианты по улучшению UX и внедрению новых функций
- Разработан пакет тестовой документации: тест-кейсы, тестовые сценарии, чек-листы

**Сайт**: [md-tests.app](https://md-tests.app/)
